# cognee-logging

Shared logging setup for the cognee-rust binaries and bindings — file rotation, Python-compatible plain-text formatter, and the default library-noise-suppressing `EnvFilter`.

Part of [cognee-rs](https://github.com/topoteretes/cognee-rs) — see the [project README](../../README.md) for an architecture overview and how the pieces fit together.
